Bienvenido a tu aplicación Flask desplegada en AWS AppRunner!

Esta aplicación se ha desplegado utilizando AWS AppRunner, un servicio completamente administrado que permite implementar rápidamente aplicaciones y API containerizadas a escala. Con AppRunner, puedes enfocarte en la lógica de tu aplicación sin preocuparte por la infraestructura subyacente.

AppRunner se encarga de todas las tareas operativas necesarias para ejecutar tu aplicación, tales como el aprovisionamiento, la gestión de escalabilidad, la configuración de redes y el balanceo de carga. Simplemente sube tu código o tu contenedor y AppRunner hace el resto.
